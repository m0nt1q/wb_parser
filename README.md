# wb_parser
    Скрипт на Python для отслеживания цен на товары Wildberries через внутреннее API.
    
    ## Функционал
    - Получение актуальной цены товара по артикулу.
    - Работа с JSON API Wildberries (v4).
    - Вывод названия товара и цены в консоль.
    ГДЕ ВЗЯТЬ cookies, headers, params?
    --------------------------------------------
    1)  Зайди на страницу товара на Wildberries.
    2)  Нажми F12 (или правой кнопкой мыши -> Просмотреть код) и перейди во вкладку Network (Сеть).
    3)  Обнови страницу (F5).
    4)  В поле поиска Filter введи слово detail или card.
    5)  Найди в списке строку, которая начинается на detail?appType=... и скопируй ее как cURL (bash)
    6) Перейди на сайт curlconverter.com и вставь скопированную ссылку
    7) Удали код парсера до строчки "data = response.json()"
    8) Вставь код скопированный из сайта curlconverter.com в парсер (в начало)
    9) Наслаждайся!

# Данный парсер можно апгрейдить, например связав его с телеграмм ботом, который в свою очередь будет присылать вам уведомление о изменении цены)
